Setup Guide

Get started with Shieldra

Step 1: Invite the Bot

Click the "Invite Bot" button above and select your server. Ensure you have the "Administrator" permission.

Step 2: Configure Permissions

Move the bot role "Shieldra" to top most role of the server.

Step 3: Setup Security

Use /setup command and set it to true to enable antinuke.

Step 4: Use Slash Commands

Type "/" in your Discord server to see all available commands. Try /help for an overview.
